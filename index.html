Let's Get Ruby and Rails Set Up

Setting up a computer to program in Ruby and Ruby on Rails requires several packages to be installed, but one of the great things about the Ruby community is the wealth of available resources to help get you started on your journey.

To start, we need to know what operating system your computer has on it.

  • My computer has Windows on it.
  • I have a Mac.
  • I am running Linux.
